# parser-medical

Parser of medical sites

### Virtual environment

Create virtual env

```bash
python3 -m venv venv
```

Activate virtual env (macos/linux)

```bash
source venv/bin/activate
```

### Installation

You can install the project dependencies using pip:

```bash
pip install -r requirements.txt
```

### Command to execute

<pre>python main.py</pre>
